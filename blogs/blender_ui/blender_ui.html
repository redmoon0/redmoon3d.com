Getting Started with Blender

This is what you will see when you start the blender. The screen in front with a beautiful scene in called splash screen. You can see that it will display your recent file and it will allow you to enter in different UI presets of blender for example General, 2D Animation, Sculpting etx.

Click on the side (Away from splash screen) and splash screen will disapper.

You can see some options on the left side. These are called tool set. Most of the time you will never click on them but you will be using them all the times with shortcut keys.

  • 1st option is to Select and tweak the objects.
  • 2nd option is to Move the 3d cursor. (You will know about 3d cursor soon)
  • 3rd option is to Move objects on X, Y or Z axis.
  • 4th option is to Move objects on X, Y or Z axis.
  • 5th option is to Rotate objects on X, Y or Z axis.
  • 6th option is to Scale objects on X, Y or Z axis.
  • 7th option is to Transform objects on X, Y or Z axis. Basicaly roatate, move and scale the object in one option.
  • 8th option is for Greece pencil. It used to draw 2d lines in the scene and also used in 2d animation but 3d artists rarely use it.
  • 9th option is for measurements. (Also rarely used)
  • Honestly I have never used the 10th option and I do not know what it is used for

Some Important Shortcuts

Key Action
G Grab and Move.
R Rotate
S Scale
If you press G and then Y it will move the selected thing in Y axis only. If you press G and then X it will move the selected thing in Y axis only. If you press G and then Z it will move the selected thing in Z axis only. Same with other 2 shortcuts (R and S).
